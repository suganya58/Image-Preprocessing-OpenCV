Text vs Non-Text Image Classifier

A simple yet powerful **Flask web application** that classifies whether an uploaded image contains readable **text content** (authentic document) or is a **non-text image** (non-authentic).  
It uses **OpenCV** and **Tesseract OCR** for image analysis and text detection.


✅ Upload any image file (JPG, PNG, etc.)  
✅ Detect whether the image contains text or not  
✅ Display prediction result with **accuracy percentage**  
✅ User-friendly drag & drop interface  
✅ Real-time image preview after upload  
✅ Built using **Flask**, **OpenCV**, and **Pytesseract**
